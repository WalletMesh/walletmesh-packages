# Function: getSystemTheme()

> **getSystemTheme**(): `"light"` \| `"dark"`

Detect system theme preference

## Returns

`"light"` \| `"dark"`

'dark' if system prefers dark mode, 'light' otherwise

## Example

```typescript
const systemTheme = getSystemTheme();
console.log('System prefers:', systemTheme);
```
